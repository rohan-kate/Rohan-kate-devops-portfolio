# Images Folder

Place your images in this folder. Here's what you can add:

## Required/Recommended Images:

### 1. Profile Photo (Hero Section) ⭐
- **File name**: `profile.jpg` or `profile.png`
- **Recommended size**: 400x400px or larger (square works best)
- **Format**: JPG or PNG
- **Usage**: Will appear in the hero section on the right side as a circular profile image
- **Note**: If not provided, a placeholder icon will be shown

### 2. Project Images (Optional but Recommended)
You can add project screenshots or images:
- **`project1.jpg`** - Online Examination System
- **`project2.jpg`** - Student Marks Management System  
- **`project3.jpg`** - ATM Simulation System
- **Recommended size**: 800x600px or similar (landscape)
- **Format**: JPG or PNG
- **Note**: If not provided, project icons will be shown instead

### 3. About Section Image (Optional)
- `about.jpg` - Personal photo or workspace image
- Can be added later if needed

## Image Naming Rules:
- ✅ Use lowercase letters
- ✅ Use hyphens or underscores (no spaces)
- ✅ Recommended formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- ✅ Keep file sizes reasonable (under 500KB each for web performance)

## How It Works:
1. **Profile Image**: Add `profile.jpg` or `profile.png` → It will automatically appear in the hero section
2. **Project Images**: Add `project1.jpg`, `project2.jpg`, `project3.jpg` → They will replace the icons in project cards
3. **Fallback**: If images are missing, the portfolio will show icons/placeholders automatically

## Quick Start:
1. Add your `profile.jpg` or `profile.png` to this folder
2. (Optional) Add project screenshots: `project1.jpg`, `project2.jpg`, `project3.jpg`
3. Refresh your browser - images will appear automatically!

## Image Optimization Tips:
- Compress images before adding (use tools like TinyPNG)
- Use JPG for photos, PNG for graphics with transparency
- Recommended dimensions:
  - Profile: 400x400px to 600x600px
  - Projects: 800x600px to 1200x800px
